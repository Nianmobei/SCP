# Sheet: 命名规则

## 内容

R1| <strong>F — Folklore（叙事 / 文化源）</strong> <strong>0 — Entity（独立实体）</strong> 异常物的命名方式以【前缀】+【别类】+【序列】命名
R2| 民俗、神话、历史叙述、象征性角色 可以被视为“一个个体”
R3| 依赖“被讲述、被理解、被相信”而存在 具备边界
R4| <strong>P — Psyche（心理 / 认知源）</strong> 可被观察、追踪、对话或攻击
R5| 由情绪、梦境、恐惧、依赖、执念生成
R6| 强烈影响意识与人格结构 <strong>1 — Object（物件）</strong>
R7| <strong>N — Natural（世界固有异常）</strong> 被明确当作“东西”使用或收容
R8| 非人造、非叙事、非单一心理 可被搬运
R9| 属于世界结构、物理法则或现实层面的异常 有明确使用方式或接触方式
R10| <strong>A — Artificial（人造 / 技术 / 概念工程）</strong>
R11| 明确或高度疑似人为制造 <strong>2 — Tool（工具 / 装置）</strong>
R12| 包括技术、概念工程、行政工具、武器化异常 <strong>功能优先于形态</strong>
R13| <strong>B— Exo / Beyond（外来 / 体系外）</strong> 确认安全专门用于达成某种效果
R14| 起源于“现实之外” 通常由机构操作
R15| 维度、世界、彼岸、河流另一侧
R16|       <strong>3 — Structure（建筑 / 固定结构）</strong>
R17|       无法轻易移动的人工或半人工结构
R19|       <strong>3 — Structure（建筑 / 固定结构）</strong>
R20|       无法轻易移动的人工或半人工结构
R22|       <strong>4 — Location（地点 / 区域）</strong>
R23|       以空间范围作为异常本体
R24|       边界不稳定或异常
R25|       人进入后才生效
R27|       <strong>5 — Phenomenon（现象）</strong>
R28|       <strong>不是东西，也不是地方</strong>
R29|       发生、扩散、消失
R30|       通常无法“收容”，只能抑制
R32|       <strong>6 — Collective（群体 / 生态）</strong>
R33|       异常以复数单位存在
R35|       <strong>7 — System（系统 / 规则）</strong>
R36|       异常本体是一套运作规则
R38|       <strong>8 — Interface（媒介 / 接口）</strong>
R39|       连接异常与现实的“端口”
R41|       <strong>9 — Mixed / Undefined（复合 / 未定）</strong>
R42|       无法在当前阶段确定其存在单位
